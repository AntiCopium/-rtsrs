## ðŸ¤– How to run
1. Install [Deno](https://deno.land/)

2. Put your stuff in .env 
```env
BOT_TOKEN=''
DEV_GUILD_ID=''
OWNER_ID=''
```

3. Run the bot with
```ps
deno task run
```
